
# Прогнозирование концентрации метана в угольной шахте

В репозитории представлен Jupyter-ноутбук с реализацией модели для прогноза концентрации метана в угольной лаве по данным промышленных датчиков.  
Цель работы — предсказывать значение концентрации метана на горизонте **15 минут вперёд** по историческим данным.

## Структура репозитория

```text
.
└── methane_forecasting.ipynb   # основной Jupyter-ноутбук с кодом
````

> При необходимости замените имя файла на фактическое название вашего ноутбука.

## Что делает ноутбук

* загружает и подготавливает данные с датчиков;
* очищает и сглаживает временные ряды (удаление выбросов, экспоненциальное сглаживание);
* формирует целевую переменную — концентрацию метана через 15 минут;
* обучает нейросетевую модель (PyTorch) для регрессии;
* оценивает качество прогноза и строит графики «факт vs прогноз».

## Требования

Минимальный набор библиотек:

* Python 3.9+
* `pandas`
* `numpy`
* `matplotlib`
* `scikit-learn`
* `torch`

При работе в Google Colab часть библиотек уже установлена.

## Как запустить

### Вариант 1. Google Colab

1. Загрузите ноутбук (`.ipynb`) в Google Drive.
2. Откройте его через Google Colab.
3. При необходимости скорректируйте путь к данным (CSV-файл).
4. Последовательно выполните все ячейки сверху вниз.

### Вариант 2. Локальный запуск

1. Клонируйте репозиторий:

   ```bash
   git clone <url-репозитория>
   cd <папка-репозитория>
   ```
2. Установите зависимости:

   ```bash
   pip install -r requirements.txt
   ```
3. Откройте ноутбук в Jupyter / VS Code и выполните все ячейки по порядку.
